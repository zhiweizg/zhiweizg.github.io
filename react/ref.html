# 使用 useRef

TIP

🌰

ref 不仅仅可以绑定在 html 标签上也可以充当做一个值，在组件中储存一个信息。

首先需从 React 中导入useRefHook 为你的组件添加一个 ref:

import { useRef } from 'react'

const ref = useRef < Number > 0

打印这个 ref 会返回一个对象

  current:{0}
    current: 0
    [[Prototype]]: Object

ref 可以指向一个数字,和 state 一样，可以保存一个数字、对象、函数、DOM 对象

与 state 不同的是 ref 是一个对象，具有 current 可以读取和修改的属性。 而且更改 ref 不可以去更新重新渲染，但是 useState 更改时可以触发重新渲染

  • ref 什么时候使用呢？
    • 使用定时器时想获得最新的数据
    • 想获得 DOM 对象，并且获取其的value或者样式
    • 计算一些 JSX 页面上一些非必需的对象

# 使用定时器获得最新的数据

若不使用 useRef，则定时器则会输出之前的数据，只会保存当时点击时的数据，并不会进行显示后面操作变化的数据。 每次点击 getCount 时，点击得到的都是一个独立的 count

🌰:

import { useState, useRef } from 'react'

function App() {
  const [count, setCount] = useState(0)
  // const ref = useRef()

  const click = () => {
    setCount((c) => c + 1)
  }

  const getCount = () => {
    setTimeout(() => {
      alert(count)
    }, 2000)
  }

  return (
    <div>
      <p>得到的count：{count}</p>
      <button onClick={click}>点击加一</button>
      <button onClick={getCount}>定时器输出</button>
    </div>
  )
}

export default App

若想得到实时的数据，只需用 ref 中的 current 属性 进行保存一下数据

🌰:

import { useState, useRef，useEffect } from 'react'

function App() {
  const [count, setCount] = useState(0)
  const ref = useRef()

  const click = () => {
    setCount((c) => c + 1)
  }

  ref.current = count
  const getCount = () => {
    setTimeout(() => {
      alert(ref.current)
    }, 2000)
  }

  return (
    <div>
      <p>得到的count：{count}</p>
      <button onClick={click}>点击加一</button>
      <button onClick={getCount}>定时器输出</button>
    </div>
  )
}

export default App

# 获取 DOM 对像

DOM 操作是 refs 最常用的方式，可以应用于各种场景

以下 🌰:

function App() {
  const ref = useRef(null)

  const handleClick = () => {
    ref.current.focus()
  }
  return (
    <div>
      <input ref={ref} placeholder="请输入"></input>
      <button onClick={handleClick}>点击输入</button>
    </div>
  )
}

export default App

# ！！！

使用 ref 操作 DOM 时，很容易造成的错误

如果你的 ref 绑定在一个{show && <p ref={ref}>Hello world</p>}state 为布尔控制的标签上

若使用 ref 去强制销毁这个标签ref.current.remove()

则会导致 setState 崩溃报错
